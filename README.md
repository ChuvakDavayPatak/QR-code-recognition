# Сканер штрих-кодов

Программа для распознавания QR-кодов на изображениях, написанная на языке программирования Python.

## Установка

1. Убедитесь, что у вас установлен Python 3.7+
2. Установите зависимости:
`pip install -r requirements.txt`
3. Для работы pyzbar на Windows может потребоваться установка Visual C++ Redistributable

## Использование

1. Запустите программу: `python Barcode.py`.
2. Нажмите "Выбрать изображение" для загрузки файла.
3. Нажмите "Сканировать" для распознавания штрих-кодов.
4. Просмотрите результаты в таблице справа.

## Поддерживаемые форматы
- Изображения: JPG, JPEG, PNG, BMP, TIFF, GIF
- Штрих-коды: QR

## Структура программы
Программа написана в среде программирования **Visual Studio 2022**.

Внутри папки *Barcode* находятся:
* изображения QR-кодов;
* **Barcode.sln** - файл проекта Visual Stuio;
* папка **Barcode** с файлами программы.

В папке *Barcode\Barcode* расположены файлы программы с расширением .py

Список основных файлов:
* **Barcode.py** - основной файл программы, в котором находится основной код Python;
* **BarcodeProcessor.py** - файл, содержащий классы для работы программы;
* **requirements.txt** - текстовый файл со списком библиотек, необходимых для работы программы.
