# Сканер штрих-кодов

Программа для распознавания QR-кодов на изображениях.

## Установка

1. Убедитесь, что у вас установлен Python 3.7+
2. Установите зависимости:
pip install -r requirements.txt
3. Для работы pyzbar на Windows может потребоваться установка Visual C++ Redistributable

## Использование

1. Запустите программу: `python Barcode.py`
2. Нажмите "Выбрать изображение" для загрузки файла
3. Нажмите "Сканировать" для распознавания штрих-кодов
4. Просмотрите результаты в таблице справа

## Поддерживаемые форматы

- Изображения: JPG, JPEG, PNG, BMP, TIFF, GIF
- Штрих-коды: QR
